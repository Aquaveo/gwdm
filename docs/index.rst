Welcome to Groundwater Data Mapper's documentation!

The Groundwater Data Mapper is a web application for uploading and visualizing groundwater  data. This app was developed using the Tethys Platform.

To learn more about the Tethys Platform see: http://docs.tethysplatform.org/en/stable/index.html.
